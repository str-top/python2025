# prompt.txt
# ==========

Ты опытный product-менеджер и UX/UI дизайнер с 10-летним опытом в e-commerce. 
Твоя задача — разработать детальное техническое задание для веб-сайта по продаже пельменей.

ОПИСАНИЕ ПРОЕКТА:

Название проекта: "Пельменная №1"

Целевая аудитория:
- Мужчины и женщины 25-45 лет
- Занятые работающие люди
- Семьи с детьми
- Студенты
- Любители домашней кухни

Основные ценности:
- Свежесть продукции (ежедневное приготовление)
- Натуральные ингредиенты
- Традиционные рецепты
- Быстрая доставка (до 60 минут)
- Доступные цены

Ассортимент:
- Классические пельмени (свинина+говядина)
- Куриные пельмени
- Пельмени с мраморной говядиной
- Веганские пельмени
- Экзотические начинки (например, с лососем)
- Сезонные предложения

Требования к функционалу:

1. Система онлайн-заказов:
   - Корзина с возможностью редактирования
   - Быстрое оформление заказа (не более 3 шагов)
   - Сохранение истории заказов
   - Интеграция с системами оплаты (карты, электронные кошельки)

2. Система доставки:
   - Расчет времени доставки в реальном времени
   - Трекинг заказа
   - SMS/email уведомления
   - Геолокация для определения зоны доставки

3. Личный кабинет:
   - Регистрация через соцсети/email
   - Бонусная программа
   - Избранные товары
   - Управление подписками

4. Админ-панель:
   - Управление товарами
   - Просмотр и управление заказами
   - Аналитика продаж
   - CRM система

5. Дополнительные функции:
   - Рецепты и идеи приготовления
   - Отзывы и оценки
   - Программа лояльности
   - Мобильная версия и PWA

Технические требования:
- Frontend: React/Next.js
- Backend: Python/Django
- База данных: PostgreSQL
- Кэширование: Redis
- Поиск: Elasticsearch
- Хостинг: облачная инфраструктура (Yandex Cloud/AWS)

Задание:
Сгенерируй детальное техническое задание включающее:
1. Юзер стори для основных сценариев использования
2. Wireframe основных страниц
3. API спецификацию для основных endpoints
4. Структуру базы данных
5. Предложения по UI/UX дизайну
6. План разработки по спринтам
7. Оценку времени и стоимости реализации

Формат ответа: детальный структурированный документ на русском языке.
